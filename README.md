# Job Portal with Application Tracking System (ATS)

A comprehensive web application for managing job postings and tracking candidate applications.

## Features

- **Job Posting**: Recruiters can post new job openings
- **Job Application**: Candidates can browse and apply for jobs
- **Candidate Dashboard**: Candidates can track their applications
- **Recruiter Dashboard**: Recruiters can manage job postings and view applications
- **Application Pipeline**: Visual pipeline to track application status
- **Admin Dashboard**: Administrator controls and management tools

## Project Structure

```
├── index.html                    # Homepage
├── post job.html                 # Job posting page

├── apply job.html                # Job application form
├── candidate dashboard.html       # Candidate dashboard
├── recrutor dashboard.html        # Recruiter dashboard
├── applcation pipeline.html       # Application tracking pipeline
├── admin dashboard.html           # Admin dashboard

```

## Technologies Used
